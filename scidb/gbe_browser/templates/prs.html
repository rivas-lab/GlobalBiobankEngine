Description of the columns

  • Trait group: trait categories used in the Global Biobank Engine. Its FAQ page has more information.
  • Trait: a trait in UK Biobank.
  • Family: it represents whether the PRS model is fit with gaussian model (for continuous response, such as standing height) or binomial model (for binary outcomes, such as disease). We use r2 and AUC as the evaluation metric for gaussian and binomial models, respectively. In this table, we display those evaluation metric computed on the hold-out test set consits of individuals of whiter British ancestry in UK Biobank cohort.
  • Geno: the predictive performence (r2 or AUC) of the predictive model with genotype features alone.
  • Covar: the predictive performence (r2 or AUC) of the predictive model with covariates (age, sex, and genotype PCs) alone.
  • Full: the predictive performence (r2 or AUC) of the full model that have both genotypes and covariates alone.
  • delta: the incremental predictive performence (r2 or AUC) of the full model against the covariate-only model, i.e. the difference between the values in "Full" and "Covar" columns.
  • # variants: the number of genetic variants included in the model
  • P (WB): P-value of the incremental predictive performance of the PRS model evaluated on the test set individuals of white British ancestry in UK Biobank. The p-value is before the multiple hypothesis correction.
  • significant?: this indicator variable shows whether the PRS model has a significant (P < 0.05/2000) incremental predictive performance.
